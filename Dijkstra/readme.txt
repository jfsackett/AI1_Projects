Mar. 5, 2014

Installation:
1) Unzip project file.
2) Open command prompt.
3) Navigate to location where project files unzipped.
4) Place javac and java in your path.
5) Execute this command to compile source:
   javac ShortestPath.java
6) Execute this command to execute program:
   java ShortestPath <input filename>
7) Shortest Path program will read input file and output results to standard output.
