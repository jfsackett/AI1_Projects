Installation:
1) Unzip project file.
2) Open command prompt.
3) Place javac and java in your path.
4) Change to src subdirectory of project.
5) Execute this command to compile source:
   javac ai1\squares\SquaresPuzzle.java (windows)
   javac ai1/squares/SquaresPuzzle.java (*nix)
6) Execute this command to execute program:
   java ai1.squares.SquaresPuzzle
7) GUI should be intuitive, command prompt contains solution move output.
