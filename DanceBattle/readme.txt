Installation:
1) Unzip project file.
2) Open command prompt.
3) Place javac and java in your path.
4) Change to src subdirectory of project.
5) Execute this command to compile source:
   javac ai1\battle\DanceBattle.java (windows)
   javac ai1/battle/DanceBattle.java (*nix)
6) Execute this command to execute program:
   java ai1.battle.DanceBattle <input filename> [-debug]
7) Dance Battle will execute and result to standard output.
