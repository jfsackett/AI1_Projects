Project Notes:
This allows the user to select the height of the Tower of Hanoi.
It also solves:
- Shakey's World
- Key In Box
- Air Cargo

Installation:
1) Unzip project file.
2) Open command prompt.
3) Place javac and java in your path.
4) Execute this command to compile source:
   javac ai1\battle\Controller.java (windows)
   javac ai1/battle/Controller.java (*nix)
5) Execute this command to execute program:
   java Controller
6) Menu will provide problem options and output to console.
