Feb. 9, 2014

Installation:
1) Unzip project file.
2) Open command prompt.
3) Place javac and java in your path.
4) Execute this command to compile source:
   javac ClosestPair.java (windows)
   javac ClosestPair.java (*nix)
5) Execute this command to execute program:
   java ClosestPair <input filename>
6) Closest Pair program will read input file and output results to standard output.
7) Output files also included (output10.txt, output100.txt, output1000.txt).
